# FreshCraft

![Screenshot](screenshot.png)

A knockoff MineCraft game created using C++ and the Vulkan Graphics and Compute API. 
It features infinite worlds, structures, placing/destroying blocks, and a first person camera.
Blocks:
- Grass
- Dirt
- Water
- Stone
- Sand
- Logs
- Leaves
- Glass
- Gold Ore
- Iron Ore
